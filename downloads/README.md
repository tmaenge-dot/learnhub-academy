# Educational Platform Downloads

This folder contains downloadable files for mobile apps and resources.

## Current Files

### Mobile Apps
- `shorthand-simplified.aab` - Shorthand Simplified Android app bundle

To add the actual file, copy it from your build folder:
```bash
cp /home/oem/Desktop/shorthand-simplified/build/shorthand-simplified.aab public/downloads/
```

## Adding New Downloads

1. Place files in this folder
2. Update the download links in the corresponding app pages
3. Ensure files are under GitHub's 100MB limit (or use Git LFS)
